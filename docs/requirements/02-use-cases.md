# 2. ユースケース

技術調査向けの試作のため、ユースケースは **1 つに集約** する。

## 2.1 想定ユーザー

自分自身（開発者）。アルゴリズムと挙動を試したいだけで、エンドユーザー像は深く想定しない。

## 2.2 ユースケース

### UC-01: ベースカラーを入れて組み合わせを確認する

- **基本フロー**:
  1. 公開 URL を開く
  2. HEX 入力欄に色を入力（またはカラーピッカーで色を選ぶ）
  3. ハードコードされたサンプルパレットから、各色彩理論ルールに対応する最近傍色がチップで表示される
  4. パレットを変えたい場合は `src/palettes.ts` を編集して再デプロイする

これが本ツールで実現する唯一のユースケース。これ以外の操作（パレット編集、保存、共有など）は提供しない。
