## 7.2 構成要素

| 要素 | 役割 |
| --- | --- |
| HEX 入力欄 | `<input type="text">` |
| カラーピッカー | `<input type="color">` |
| パレット表示 | サンプルパレットの色を横並びチップで表示 |
| 結果セクション | ルール名見出し + チップ列 |
| チップ | 色のスウォッチ + HEX 値 |

特別なコンポーネント設計はせず、`src/ui.ts` の中で素直に DOM を組み立てる。
