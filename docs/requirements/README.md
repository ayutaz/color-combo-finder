# 要求定義書

色の組み合わせ計算・可視化ツール（Color Combo Finder）の要求定義一式。

## このプロジェクトの位置づけ

**技術調査・試作**が主目的。商用品質のフルスペックなツールを作るのではなく、

- 色彩理論に基づく組み合わせ計算アルゴリズム
- パレットへの近傍マッピング（CIEDE2000 など）
- ブラウザでの軽量な可視化

これらを **最小コードで動かして理解する** ことに価値を置く。

そのため要求定義もスコープを絞り、**Must 項目を最小限に** している。「あったら便利」な機能は意図的に Won't に倒している。

## 用語

| 用語 | 意味 |
| --- | --- |
| ベースカラー | ユーザーが指定する基準色。組み合わせ計算の起点。 |
| パレット | 事前に登録された数色のセット。組み合わせの候補集合。 |
| 組み合わせ（コンボ） | ベースカラーとパレット内の色で構成される N 色の集合。 |
